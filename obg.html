Dados do Projeto

PROJETO DE CONCLUSÃO DE CURSO DEVSTARS
(MÓDULO FUNDAMENTOS WEB - PROF. DANIEL QUADROS)

O objetivo deste projeto é desenvolver uma loja virtual simples que importe produtos do Mercado Livre e permita adicionar itens a um carrinho de compras. O foco é praticar a manipulação de APIs, a manipulação do DOM e o uso de localStorage para gerenciar o estado do carrinho de compras.

Desenvolvido por Marcos Aurélio de Freitas, este projeto faz parte do módulo Fundamentos Web da PCD Web DevStars e foi realizado no ano de 2024. A data de conclusão é 07/08/2024 e a modalidade de ensino foi à distância via Microsoft Teams.

O projeto foi orientado e avaliado pelo professor Daniel Quadros, Mestre em Fullstack.
